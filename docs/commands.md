# Commands

Prefix `$`

## The `help` command
__No arguments__
> $help

`fruit` __argument__
> $help fruit
Embed of specific command

## The `fruit` command
__No arguments__
> $fruit
Embed of fruit list
`<1-50>` `<fruit>`
> $fruit 10 watermelon

## The `stonks` command
`<any>`
> $stonks
Stonks picture with any string

## The `profile` command
__No arguments__
> $profile
